Ramsomwer

What is it?
Ransomware is a kind of program which is installed in a device using methods of contact or communication that restricts access to certain parts or files of the infected operating system and demands a ransom or a certain amount of money in exchange for removing this restriction. Some types of ransomware encrypt operating system files rendering the device unusable and forcing the user to pay the ransom.

How does ransomware work?
Ransomware uses asymmetric encryption. This is cryptography that uses a pair of keys to encrypt and decrypt a file. The public-private pair of keys is uniquely generated by the attacker for the victim, with the private key to decrypt the files stored on the attackerâ€™s server. The attacker makes the private key available to the victim only after the ransom is paid, though as seen in recent ransomware campaigns that are not always the case. Without access to the private key, it's nearly impossible to decrypt the files that are being held.

Characteristics of Ransomware.
- Crash of the entire operating system.
- Data restriction.
- attached program in mail files.
